# Loom

A game engine for on-chain worlds, supplied as a protocol: an ECS runtime, a tick
scheduler, and a compute bridge on Solana, so every world shares one state model.

Early scaffold. Needs Node 22.6+ (native TypeScript), Rust, and pnpm; the on-chain
program also needs the Solana and Anchor toolchains. See `PROJECT_SPEC.md` for the
design.
